# 📊 项目管理平台 Demo

一个简洁高效的项目管理演示平台，展示核心功能：看板视图、列表视图、任务管理等。

## 🌐 在线访问

**GitHub Pages 链接（推荐）:**
```
https://huicent.github.io/project-manager-demo/
```

## ✨ 核心功能

### 📋 看板视图
- 三列布局：待处理、进行中、已完成
- 点击任务卡片快速切换状态
- 任务数实时统计

### 📝 列表视图
- 表格形式展示所有任务
- 支持按优先级、状态、分配人查看

### 📊 项目统计
- 实时进度条（可视化进度）
- 任务数统计（总任务、进行中、已完成、待处理）
- 多项目快速切换

### ✏️ 任务管理
- 快速添加新任务（输入 + Enter）
- 优先级设置（高/中/低）
- 分配人和截止日期跟踪

## 🚀 快速开始

### ✅ 推荐：在线访问
直接打开：https://huicent.github.io/project-manager-demo/

### 本地运行
```bash
# 克隆仓库
git clone https://github.com/Huicent/project-manager-demo.git
cd project-manager-demo

# 启动服务（任选其一）
python3 -m http.server 8080
# 或
python -m SimpleHTTPServer 8000
# 或
npx http-server

# 访问 http://localhost:8080
```
